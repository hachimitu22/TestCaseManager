# TestCaseManager (tcm)

TestCaseManager は、テストケースの作成・管理を支援する OSS ツールです。

## 特徴

- CLI と Web UI の両方で利用可能
- テストケースをファイルとして管理し、Git で差分管理しやすい
- 出力は XML 形式に統一

## クイックスタート（予定）

```bash
# 作業フォルダの初期化
tcm init <dir>

# サーバー起動（デフォルト: 3000）
tcm server -p <port>
```
